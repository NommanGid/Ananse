1. What HTML Is

‚úÖ What is HTML?

HTML (HyperText Markup Language) is used to build the structure of a webpage.

Think of a website like a house üè†:

  • HTML = walls, doors, rooms
  • CSS = paint, decorations
  • JavaScript = lights, buttons, actions

‚úÖ What HTML Does (Structure, Not Style)

HTML tells the browser:

  • What is a heading
  • What is a paragraph
  • What is an image or link

‚ùå HTML does not decide colors or layout ‚Äî that's CSS.

‚úÖ How Browsers Read HTML

  • Browser reads HTML top to bottom
  • Builds page structure
  • Displays content visually

‚úÖ HTML vs CSS vs JavaScript

Language Job
HTML Structure
CSS Style
JavaScript Behavior
