‚úÖ What is JavaScript?

JavaScript (JS) makes websites interactive.

Think of a website like a robot ü§ñ:

  • HTML = body

2. Variables & Data Types

‚úÖ Variables

Variables store data.

‚úîÔ∏è Use let and const ‚ùå Avoid var

‚úÖ Primitive Data Types

true / false // boolean
null         // empty on purpose
undefined    // not assigned yet

‚úÖ typeof

‚úÖ Type Coercion (Basics)
